# 📖 Project Overview

The sinking of the **RMS Titanic** remains one of the most well-known maritime disasters in history. This project presents a comprehensive **Exploratory Data Analysis (EDA)** of the Titanic dataset to uncover meaningful insights into passenger demographics, travel information, and the factors that influenced survival.

The analysis includes data cleaning, missing value treatment, feature engineering, statistical exploration, and informative visualizations using Python.

---

# 🎯 Objectives

- Understand the structure of the dataset
- Perform data cleaning and preprocessing
- Handle missing values
- Explore categorical and numerical variables
- Detect outliers
- Analyze factors affecting passenger survival
- Perform feature engineering using Age Group Binning
- Visualize relationships among variables
- Summarize key insights from the dataset

# 📈 Key Findings

- Female passengers had significantly higher survival rates than males.
- First-class passengers were more likely to survive than passengers in lower classes.
- Younger children exhibited higher survival rates than older passengers.
- Higher ticket fares were generally associated with better survival outcomes.
- Most passengers belonged to the Young Adult age group.
- Missing values in the Age and Embarked features were successfully handled using appropriate imputation techniques.
- Numerical features showed relatively weak correlations, indicating limited multicollinearity.
